REST API для учета продаж коллекционных монет из драгоценных металлов

Через API можно добавлять, удалять, обновлять и считывать драгоценные коллекционные монеты, которые есть в базе. Монеты сделаны из драгоценных металлов, при этом имеют коллекционную ценность. Через API можно рассчитать коллекционную составляющую цены монеты.
